# Module Grid

Working reference for how modules are held and placed on the ship. In-flux; not canon.

Built on the Matchbox grid system (mechanical only: placement, footprints, rotation, adjacency/row/column queries). SpaceMatch layers the rules on top; Matchbox stays game-agnostic.

## Rules

- **No separate inventory.** Modules live on the ship's module grid — there is no off-grid backpack to hoard unused modules in.
- **Forced placement.** A module you acquire goes onto the grid.
- **Won't fit → discard or make it work.** If an acquired module doesn't fit, you either discard a module or rearrange the grid to make room. You can't keep what you have no space for.
- **Storage (if ever added) is the same system.** Optional future storage would be the same module grid as the ship, just without bonuses — not a separate inventory system.
