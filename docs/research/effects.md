# Effects

- **Damage-over-time** — poison / burn / bleed
- **Skip-turn / can't-act** — stun / paralyze / freeze / sleep
- **Heal-over-time** — regen
- **Defense / absorb** — block / shield
- **Stat-up** — strength, haste
- **Stat-down** — weak, slow
- **Damage-taken-up** — vulnerable / mark
- **Damage-taken-down** — mitigate / resist
